# autoheal

CLI tool that automatically fixes code based on failing tests.

<br/>

## How does it work?
<br>

<br/>

## How to use

In your project directory, run:
```
npx autoheal
```

Uses OpenAI's GPT-3.5-turbo or GTP-4 APIs. [Requires OpenAI API key.](https://beta.openai.com/)
